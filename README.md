## Installation

```sh
git clone https://github.com/thanhvule0310/taki-server
cd taki-server
pnpm install
```

Create `.env` file and put your configrations into it. Refer to the .env.example file for more information.

## Run

```sh
pnpm run start:dev
```

### Required

- NodeJS
- MongoDB

## Features

- RESTful API
- NestJS
- Swagger
- JWT
- MongoDB via Mongoose
- NodeMailer
- Google Auth
- Facebook Auth
